UI/UX DESIGN

Design for Anjungan Desa

Created a user-friendly interface and visual concept for the Anjungan Desa Glagahwero platform. The design focused on accessibility, clear navigation, and presenting village information in a modern yet simple way, making it easier for the community to access public services digitally.

WEB DESIGN

Landing Page Design for Belankon E-Commerce

Designed a modern and responsive landing page for Belankon, an e-commerce platform. The layout highlights products with clean visuals, engaging call-to-actions, and a user-friendly flow to improve shopping experience and brand presence online.

DESKTOP APP

Java Cashier Application

A Java-based desktop application built for cashier operations. Features include cashier login, product & stock CRUD, and a transaction module to process purchases. Integrated with a phpMyAdmin database and supported by triggers that automatically update stock after each transaction, the system is designed to streamline and support basic retail operations effectively.
